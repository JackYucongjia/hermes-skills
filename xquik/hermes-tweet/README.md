# Hermes Tweet

Hermes Tweet is an ASK-compatible wrapper for the native Hermes Agent plugin
that gives agents a safe X/Twitter route through Xquik.

Install the native plugin:

```bash
hermes plugins install Xquik-dev/hermes-tweet --enable
```

Set `XQUIK_API_KEY` in the Hermes runtime environment before authenticated
reads or approved actions. Do not paste keys into chat.
